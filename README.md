# Bible Books Tracker
An application to track your Bible reading progress.
Created with Next.js

## Getting Started

**Running locally:**

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000)

**Running on Vercel:**

Open [https://bible-books-tracker.vercel.app](https://bible-books-tracker.vercel.app)
