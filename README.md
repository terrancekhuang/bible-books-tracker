# Bible Books Tracker

An application to track your Bible reading progress.
Created with Vite

## Getting Started

**Running locally:**

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000)
